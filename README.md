# 🐍 VENOM RUSH — Snake Battle Royale

> A multiplayer snake battle royale unlike any other.
> **INFECT · EVOLVE · DOMINATE**

## 🎮 Play Now
**[▶ Play at venomrush.vercel.app](https://venomrush.vercel.app)**

## ✨ Unique Features
- 🧬 **Infection Mode** — one snake spreads virus to all
- 🌀 **Wormholes** — teleport portals across the map
- 🔬 **Evolution** — kill infected bots to gain traits
- ☣️ **Toxic Food** — risk vs reward mechanic
- 💉 **Venom Puddles** — boost to leave slowing trails

## 🕹️ Game Modes
Survival · Infection · Deathmatch · King Zone · Battle Royale · 2v2 Teams

## 🚀 Run Locally
```bash
npm install
npm run dev
```

## 🌍 Deploy
```bash
npm install -g vercel && vercel --prod
```
